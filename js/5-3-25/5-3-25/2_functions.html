3 Ways to declare a function

1. function decleration

Syntax: 

function function_name (para){
                //code    
            }
                call a function:
        
                function_name(arg)

2. function expression - 

            const function_name = function(para){
                //code
            }

            //call
            function_name(arg);

3. arrow function 

            const function_name = (para) =>{
                //code
            }
            //call
            function_name(arg)
